trimPolyfill

该trim()方法从字符串的两端删除空格。在这个上下文中的空格是所有的空格字符（空格，制表符，无中断空格等）和所有行终止符（LF，CR等）

var str = " ioing "; str = str.trim(); console.log(str); // "ioing"

splitCells

该splitCells()方法和split()方法相似，都是通过将字符串分隔成子字符串将对象分割成字符串数组，但该方法会忽略字符中被引号包裹的部分。

var str = "abc,def'g,h'"; console.log(str.split();); // ["abc", "def'g", "h'"] console.log(str.splitCells();); // ["abc", "def'g,h'"]

repeatPolyfill

该repeat()方法构造并返回一个新的字符串，其中包含指定数量的被调用的字符串的副本，并且连接在一起。

/** * str: String * count: Number */ let resultString = str.repeat(count);

例子

'abc'.repeat(-1); // RangeError 'abc'.repeat(0); // '' 'abc'.repeat(1); // 'abc' 'abc'.repeat(2); // 'abcabc' 'abc'.repeat(3.5); // 'abcabcabc' (count will be converted to integer) 'abc'.repeat(1/0); // RangeError ({ toString: () => 'abc', repeat: String.prototype.repeat }).repeat(2); // 'abcabc' (repeat() is a generic method)

paramsToObject

该paramsToObject()方法构造并返回一个新的字符串，其中包含指定数量的被调用的字符串的副本，并且连接在一起。

var param1 = "id/100/page/15/limit/20"; var obj1 = param.paramsToObject(); // { id: 100, page: 15, limit: 20 } var param2 = "id=100&page=15&limit=20"; var obj2 = param.paramsToObject(); // { id: 100, page: 15, limit: 20 } var param3 = "id:100,page:15,limit:20"; var obj3 = param.paramsToObject(); // { id: 100, page: 15, limit: 20 }

staticAnalysis

该staticAnalysis()方法对字符进行 javascript 静态分析，返回该字符串的 javascript 语法分析类型。

值 描述
number 该字符串为数字类型
function 该字符串为函数类型
object 该字符串为对象类型
token 该字符串为 javascript 关键字
variable 该字符串为一个变量名
unknown 该字符串为未知类型
"count".staticAnalysis(); // "variable" "String".staticAnalysis(); // "function" "Array".staticAnalysis(); // "function" "document".staticAnalysis(); // "object" "document.body".staticAnalysis() // "object" "var".staticAnalysis(); // "token" "for".staticAnalysis(); // "token"

说明：如果该字符解析时默认有值则返回该结果值类型，如 document，如果该值为值变量则返回 "variable"。
