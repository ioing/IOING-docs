href 除了跳转链接外还能够跳转到模块

指令 值 描述
href [url] 页面跳转
href #[module]/[param] 模块转场

使用 href 进行页面跳转，其属性值为 URL

data-event

注意

需要注意的是捕捉模块跳转和链接跳转的事件是不同的，链接跳转时应监听 'href' 事件捕捉，而模块跳转则依循transform指令，监听 'transform' 事件捕捉
