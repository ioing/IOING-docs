数据

为组件定义内部数据源

指令 值 描述
data:[name] [data-name] 定义组件数据源（命名且指向外层数据源）
[attr=value] [String] 通过设置元素属性也可以在数据源种体现该值

组建内获取组件数据源值

假设模块有数据源对象 setting，而此时组件需要接入该对象

在该组件中的脚本通过下面的方式读取该值

var setting setting = scope.config // 定义在组件的数据源通过 scope 对象获取 setting = scope.setting // 该 scope 中的 setting 为模块数据源

在该组件中的模版可以通过下面的方式读取该值

{config.name}

组建内获取组件 Attr 属性值

为下面的组件设置一个 value 属性且给值 40

在该组件中的脚本通过下面的方式读取该值

var val val = scope.val // 定义在组件的数据源通过 scope 对象获取 val = components.attr('val') // 也可以通过传统方式获取，components 为组件自身元素

在该组件中的模版可以通过下面的方式读取该值

{val}
