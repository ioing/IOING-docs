@class

定义一个 CSS 属性继承类

指令 值 描述
@class name 定义一个类

使用@class引用类可继承此类中定义属性

定义一个名为 commonIcon 的类，并为这个类设定基本描述属性

@class commonIcon (radius) { border: 1px solid rgba(0,0,0,.15); border-radius: [radius]dp; background-size: 102% auto; background-repeat: no-repeat; background-position: center; background-clip: border-box; }

引用类 commonIcon 并传入一个参数

.icon { width: 72dp; height: 72dp; @class: commonIcon(6); }

经过引擎解释后的结果为

.icon { width: 72dp; height: 72dp; border: 1px solid rgba(0,0,0,.15); border-radius: 6dp; background-size: 102% auto; background-repeat: no-repeat; background-position: center; background-clip: border-box; }
