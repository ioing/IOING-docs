@extend

继承指定选择器下的属性集合

指令 值 描述
@extend selector 继承指定属性集合
继承能力能够使高度耦的属性进行重复使用，极高的提升开发速度，对便于后期管理维护具有重要意义，该方法和 # @class 类似

类 .btn 描述一个按钮的基本外观，在需要基于这个外观进行定制个性化按钮的类中可通过 @extend 来继承这个基本外观属性

.btn { padding: 4dp 20dp; min-height: 32dp; color: #fff; background: [common-background]; border-radius: 6dp; } .btn-blue { @extend: .btn; background: blue; }

.btn-blue 类经过解释后变为

.btn-blue { padding: 4px 20px; min-height: 32px; color: #fff; border-radius: 6px; background: blue; }

以往我们在进行 CSS 描述抽象化的时候都会使用在一个 DOM 节点上增加 N 多个类名的方法将 CSS 拆分，这样的后果是 DOM 结构被侵入变得难以维护，而通过动态继承则能够更好的专注维护 CSS 样式表的构建
