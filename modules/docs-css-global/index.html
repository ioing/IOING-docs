@global

全局变量声明功能

指令 值 描述
@global name:value 定义一组全局变量声明

使用@global定义的变量可以被跨文件读取，当一组全局变量被定义时可在任何的后续 CSS 文件中读取定义的变量

如在模块 frameworks 中进行 global 定义如下全局变量

@global { header-height: 60dp; max-width: 1000dp; common-color: blue; }

在模块 home 的 CSS 资源中来读取上面定义的全局变量

div { margin-top: [header-height] max-width: [common-width]; width: 100%; height: 100%; color: [common-color]; }

上例在浏览器中的实际解释结果为

div { margin-top: 60dp; max-width: 1000dp; width: 100%; height: 100%; color: blue; }

全局变量一般定义在主模块中，如果你定义在子模块中且该子模块当前还未载入，这时如果你当前有依赖该子模块的全局变量时该变量将无法读取

基于上述特性我们可以通过 @if 条件判断 来检测某个全局变量是否存在来判断该模块是否已经被加载
