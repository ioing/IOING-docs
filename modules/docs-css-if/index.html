@if

若假设条件成立则继续编译条件内 CSS，表达式内支持基本的 javascript 运算

指令 值 描述
@if (表达式) 表达式成立时则执行包含内容，不成立则忽略包含内容
@if 中的逻辑表达式符合#标准-逻辑运算中的定义

根据满足条件渲染 CSS，以自定义 CSS 变量为例

@var { max-width: 25dp; common-color: blue; } @if ( max-width ) { div { height: 20dp; background: [common-color]; border-top: solid [max-width] red; } }

根据不同设备操作系统实现不同样式输出

@if ( device.os.ios ) { header { position: fixed; top: 0; width: 100%; z-index: 10000; height: [header-height]; background: rgba(255,255,255,.8); background-clip: content-box; border-bottom: 1px solid rgba(0,0,0,.1); backdrop-filter: blur(20dp); } }

根据模版数据源做逻辑判断

@if ( result.length ) { .gird { width: [result.length]%; } }
