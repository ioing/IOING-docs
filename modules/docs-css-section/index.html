@section

定义 CSS 的作用域，即产生 CSS命名空间

指令 值 描述
@section selecter 创建一个命名空间选择器

对 header 节点下样式设定一个命名空间

@section ( header ) { h1 { margin: 15dp 0; color: #333; font-size: 18dp; } .a, .b, .c, .d { color: red; font-size: 12dp; } .n { width: 50%; } }

这段代码实际在浏览器解析前被转换为

header h1 {...style} header .a, header .b, header .c, header .d { color: red; font-size: 12dp; } header .n { width: 50%; }

通过关键字this设定 section 自身样式

@section ( .link ) { this { color: #000; font-size: 12dp; } this a { color: blue; } this.green { color: green; } }

这段代码实际在浏览器解析前被转换为

.link { color: #000; font-size: 12dp; } .link a { color: blue; } .link.green { color: green; }

在一个 section 块中创建子 section

@section ( header ) { h1 {...style1} @section (.nav) { this {...style2} nav ul {...style3} nav ul li {...style4} } }

这段代码实际在浏览器解析前被转换为

header h1 {...style} header .nav nav ul {...style} header .nav {...style} header .nav nav ul li {...style}

嵌套使用是被允许的，但我们并不建议这样规划你的样式表
