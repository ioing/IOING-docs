@var

局部变量声明功能

指令 值 描述
@var [name : value] 定义一组局部变量

使用@var指令和我们平时使用var在 javascript 中定义变量类似，可以将变量定义在不同的作用域中

文件作用域

当直接定义在 CSS 文件的根时，其值的作用域为当前文件中

@var { common-width: 25dp; common-color: blue; } div { width: [common-width]; height: 20px; background: [common-color]; }

区块作用域

当定义在 section 区块中时，其值作用域为当前区块内#了解 section 区块

@var { height: 25dp; } @section (.main) { @var { width : 90vm; height : 50hm; color : red; } div { margin : @([height] - 20dp + 1px); width : [width]; height : [height]; color : color; } }
