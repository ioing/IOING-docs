引入模版片段

使用 include 可以引入 #资源配置 中（或通过 #template 定义的）模版资源

指令 值 描述
name [ template name ] 载入指定 name 的模版片段
data [ data name ] 载入指定 data 数据源

在 include 标签中还有第二个指令 data，该指令能够为引入的模版指定引用的数据源，一般这是不必要的，因为 include 本身就能够继承他的父级数据源，自定义引用的目的不在于制定数据源而是通过 #data指令重命名数据源
