循环标签

遍历指定数据源，循环生产模版片段。

指令 值 描述
data/origin [origin] 指定循环数据源
as 无 关键词，需与value、key 联合使用
value 无 声明结果值的变量名称
key 无 声明键值的变量名称
缺省 'as value key' 时，value 与 key 默认取名为 $value、$index

循环数据源对象 list， 并以 value 为结果值名称，以 key 为键值名称。

  • {key} : {value}

    • 若此事例循环数据源对象 list 下的子对象 list.names，且没有使用 as 关键词

  • {$key} : {$value}

    • 下面是个完整示例，在 #模块配置 中配置据源为

    define(function (require, module, exports) { module.exports = { resources: { data: { list: [ {"firstname": "Hall", "lastname": "James"}, {"firstname": "Smith", "lastname": "John"}, {"firstname": "Baker", "lastname": "David"}, {"firstname": "Cook", "lastname": "Daniel"}, {"firstname": "Miller", "lastname": "Michael"} ] ... } }, config: { data: ['list'] ... } } })

    输出索引及完整姓名

  • {i}. my name is {val.firstname} {val.lastname}

    • 运行结果如下

    [ {"firstname": "Hall", "lastname": "James"}, {"firstname": "Smith", "lastname": "John"}, {"firstname": "Baker", "lastname": "David"}, {"firstname": "Cook", "lastname": "Daniel"}, {"firstname": "Miller", "lastname": "Michael"} ]
    • {i}. my name is {val.firstname} {val.lastname}
