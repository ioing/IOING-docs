#shadow-root

通过使用 shadow 能很好的把 CSS 模块化，并解决 CSS 耦合

my color is black
my color is black
my color is blue

在 shadow 标签内的节点将脱离主文档存在，其产生两个特性：

  • 一、影子节点内不受外部样式干扰。
  • 二、影子节点内的样式不会干扰到外部。
产生这两个特性的前提是你的浏览器需支持创建shadowroot，当你的设备不支持时第一条特性将不成立，如果你需要兼容到不支持设备上，你需要注意模块的 CSS 样式避免被内部继承。
