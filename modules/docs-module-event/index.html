transformstart 转场开始事件

进入该模块时发生转场动画之前

属性 类型 默认值
transformstart Function 无
export default { resources: { ...resources }, config: { ...config }, events: { transformstart: function () { ... }, ... } }

在转场开始时该事件函数将被执行，若该函数return false则转场将被即刻终止

transformsend 转场结束事件

进入该模块时发生转场动画之后

属性 类型 默认值
transformend Function 无
export default { resources: { ...resources }, config: { ...config }, events: { transformsend: function () { ... }, ... } }

当模块被设置动画时，动画执行结束后将执行该配置项的函数，若无动画则即刻执行该配置项的函数

preload 模块加载数据源之前事件

进入该模块时发生数据源加载之前执行此事件

属性 类型 默认值
preload Function 无
export default { resources: { ...resources }, config: { ...config }, events: { preload: function () { ... }, ... } }

该配置项的执行时机与 transformsend 相近

load 模块加载完毕后事件

进入该模块直到该模块完整加载完毕后执行此事件

属性 类型 默认值
load Function 无
export default { resources: { ...resources }, config: { ...config }, events: { load: function () { ... }, ... } }

该事件在模块载入完毕（不包含异步组件部分）时执行

loading 模块动画事件

属性 类型 默认值
loading Function 无

此事件被设定后，引擎默认加载的 loading 将失效，使用该设定可以自定义模块的加载动画

export default { resources: { ...resources }, config: { ...config }, events: { loading: function (id, display) { ... }, ... } }

该事件函数可接收两个入参模块id和可见性display，如果该选项未被设定同时也未被设定为false那么该选项将跟随主模块的该项设定，如果主模块设定了 loading 事件，那么该模块的 loading 也将执行该设定
