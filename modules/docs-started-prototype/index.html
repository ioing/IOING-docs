IOING 控制了应用从载入到展示的整个过程，被载入的模块文件经过语法解释器生成准浏览器识别文件，而这一切都是在前端进行处理的。

开发目录

默认开发目录中包含如下文件

文件 描述
index.html 程序入口文件，在开发中该文件一般不需要修改
ioing.js 该文件为引擎核心文件
ioing 引擎开发目录，可删除
modules 模块目录，放置开发的模块
components 组件目录，放置开发的组件

目录ioing是引擎的源代码目录，目录包含以下文件

文件 描述
unify.js 环境统一脚本，主要对浏览器进行功能检测和兼容扩展
application.js 负责应用程序的初始化和启动
css.js CSS 语法解释器
dom.js DOM 语法解释器
fetch.js 模块资源异步加载器
loader.js 模块默认 loader
move.js CSS3 操作方法
promise.js 异步处理库
proto.js 扩展库
query.js DOM Query 库
sandbox.js 沙盒化处理器
scroll.js Gpu 加速滚动库
touch.js 多手势操作库
transform.js 路由和模块转换管理
template.js 模块视图管理
